# 数据资产管理平台

基于 Vue 3 + Vite + JavaScript + Element Plus 构建的政务数据资产管理平台。

## 技术栈

- **前端框架：** Vue 3 (Composition API)
- **构建工具：** Vite
- **UI 组件库：** Element Plus
- **路由管理：** Vue Router 4
- **状态管理：** Pinia
- **HTTP 请求：** Axios
- **数据可视化：** ECharts

## 功能模块

### 数据资产
- **数据目录** - 查看所有数据资产目录
- **资产登记** - 新增数据资产（分步表单）
- **资产审核** - 审核提交的资产登记申请
- **资产管理** - 增删改查、批量操作

### 数据共享
- **共享申请** - 申请使用数据资产
- **共享审批** - 审批共享申请
- **共享记录** - 查看历史共享记录

### 数据安全
- **安全策略** - 配置数据安全策略
- **安全日志** - 查看操作审计日志

### 系统管理
- **用户管理** - 用户 CRUD、批量删除
- **角色管理** - 角色权限配置
- **部门管理** - 部门组织结构管理

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
